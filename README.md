# 杭州地铁购票APP

##  2017-01

* 正常的完整购票流程（当然支付是做作样子，没写接口）
* 目前只支持同线路站点，站名首字母大写分类
* 支持查询购票记录，包括未取票的记录和已取票的历史记录
* 添加了下拉刷新
* 用Laravel写的后台服务端作为数据交互
* 对用户认证功能设计了基于OAuth2协议的认证（测试）。
